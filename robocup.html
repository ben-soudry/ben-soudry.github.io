Robocup Junior Soccer

For this project, I worked with a friend to design, build and program a robot to compete in the Robocup Junior Soccer competition. In this competition, robots play a game of fully autonomous, 2v2 soccer. The game is played with an infared emitting ball, which allows the robots to detect it using basic sensors.

We won the Judges Award at Robocup Junior USA 2015 for our robot's design, our poster and documentation. For more information about the competition, click this link: Robocup Junior Soccer

Robot Design

Before building the Robot, we created a detailed computer model using Autodesk Inventor. The design uses a combination of 3d-printed, laser-cut, and standard parts
